# Big Data Analytics for Customer Behavior using Apache Spark and Machine Learning

## Project Overview

This project applies Big Data Analytics techniques using Apache Spark (PySpark) to analyze customer purchasing behavior from an online retail transactional dataset.

The project combines:

- Descriptive Analytics
- Diagnostic Analytics
- Customer Segmentation
- Predictive Analytics

to generate actionable business insights for customer intelligence and marketing optimization.

---

## Technologies Used

- Apache Spark (PySpark)
- Python
- Pandas
- Matplotlib
- Scikit-Learn
- MLlib

---

## Dataset

Online Retail transactional dataset containing:

- Invoice information
- Product descriptions
- Quantity purchased
- Unit price
- Customer IDs
- Countries

---

## Project Workflow

### 1. Data Preprocessing

- Missing value handling
- Removing invalid transactions
- Feature engineering
- Timestamp transformation

### 2. Exploratory Data Analysis

- Revenue by country
- Monthly sales trends
- Peak purchasing hours
- Correlation analysis

### 3. Customer Segmentation

RFM Analysis:

- Recency
- Frequency
- Monetary

K-Means clustering was applied to classify customers into:

- High-value customers
- Regular customers
- At-risk customers

### 4. Predictive Analytics

Decision Tree Classification was used to predict high-value customers based on:

- Total quantity purchased
- Average product price
- Total spending
- Number of transactions

---

## Results

- Decision Tree Accuracy: ~95%
- Effective customer segmentation using K-Means
- Identification of high-value and at-risk customers
- Revenue and seasonal trend insights
